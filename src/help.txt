ELM TEXT EDITOR ============================================================ [v0.1]

A. PERKENALAN APLIKASI
ELM Text Editor merupakan sebuah software yang memungkinkan pengguna untuk membuka, 
melihat, dan mengedit file plain text atau teks biasa. Software ini dibuat 
menggunakan bahasa pemrograman C/C++. 

B. FUNGSIONALITAS
ELM Text Editor memiliki beberapa fitur
1. Copy     : Menyalin suatu kata atau kalimat dari satu tempat menuju ke 
              tempat lainnya.
2. Paste    : Menempelkan atau memindahkan kata atau kalimat yang sudah di copy.
3. Find     : Mencari suatu kata atau kalimat.
4. Replace  : Merubah kata yang ingin diganti.
5. Open     : Membuka file dokumen yang tersimpan di komputer.
6. New      : Mebuka dokumen lembar kerja baru.
7. Save     : Memperbarui file yang terakhir disimpan dengan konten terbaru.
8. Save As  : Menyimpan file baru atau menyimpan file yang sudah ada ke lokasi baru
              dengan nama yang sama atau berbeda.
9. Quit     : Keluar dari program.

    B.1 HOTKEYS
    Berikut beberapa shortcut yang dapat digunakan dalam ELM Text Editor
    1. CTRL + C : Copy
    2. CTRL + V : Paste
    3. CTRL + F : Find
    4. CTRL + R : Replace 
    5. CTRL + O : Open
    6. CTRL + Q : Quit
    7. CTRL + N : New
    8. CTRL + S : Save
    9. CTRL + X : Save As
